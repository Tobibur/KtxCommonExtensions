# KtxCommonExtensions

## Add the dependency

### Add it in your root build.gradle at the end of repositories:

```Grale
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

###  Add the dependency

```Gradle
dependencies {
	implementation 'com.github.Tobibur:KtxCommonExtensions:1.0.1'
}
```
